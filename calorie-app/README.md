# Calorie Buddy

Лёгкий оффлайн-first помощник для учёта калорий. Хранит записи в `localStorage`, позволяет задавать дневную цель и быстро добавлять блюда.

## Запуск

1. Открой папку `calorie-app`.
2. Дважды щёлкни `index.html` (или запусти любой статический сервер, например `pwsh -c "cd calorie-app; python -m http.server"`).

Работает полностью локально и не требует сборки. Состояние автоматически сохраняется в браузере.
