# OrgChat Web

Next.js 14 (App Router) клиент для внутреннего чата.

## Что уже сделано
- Tailwind + Zustand + Socket.IO клиент.
- Базовый макет рабочего пространства (каналы + чат).
- Общая кнопка берётся из `@org-chat/ui`.

## Локальный запуск
```bash
pnpm install
pnpm --filter org-chat-web dev
```

Переменные среды берутся из `.env` в корне монорепо (`env.example` доступен для копирования).
