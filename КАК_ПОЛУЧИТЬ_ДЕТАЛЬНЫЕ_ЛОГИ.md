# Как получить детальные логи ошибки компиляции

## Проблема
Деплой на Render всё ещё падает с ошибкой компиляции Kotlin, но в логах не видно точной причины.

## Что нужно сделать

### Вариант 1: Скопировать полные логи из Render

1. В Render Dashboard откройте вкладку **"Logs"** (не "Events")
2. Найдите строки с `[ERROR]` или `Compilation error`
3. Скопируйте **все строки**, начиная с `> Task :server:compileKotlin FAILED` до конца ошибки
4. Отправьте мне эти логи

### Вариант 2: Использовать "Problems Report"

В логах видно:
```
[Incubating] Problems report is available at: file:///app/build/reports/problems/problems-report.html
```

Но этот файл недоступен через веб-интерфейс Render.

### Вариант 3: Проверить локально

Попробуйте собрать проект локально:

```bash
cd server
../gradlew :server:compileKotlin --stacktrace
```

Если ошибка воспроизводится локально, скопируйте вывод команды.

## Что я проверяю сейчас

Проверяю возможные проблемы:
- Использование `epochSecond` (может быть проблема с версией Kotlin)
- Импорты `kotlinx.serialization`
- Другие синтаксические ошибки

## Временное решение

Если не можете получить логи, попробуйте:
1. Открыть Render Dashboard → Logs
2. Прокрутить вниз до строк с `[ERROR]`
3. Скопировать все красные строки
4. Отправить мне
